## Install

```bash
npm install vue3-select
```

Then, import and register the component:

```js
import Vue from 'vue'
import VueSelect from 'vs-vue3-select'
import 'vs-vue3-select/dist/vs-vue3-select.css'

app.component('v-select', VueSelect)
```
